# knowing-kiss-8592
KindMeal is Malaysia's (and possibly the world's!) pioneering meat-free lifestyle platform. We provide an effective platform to promote compassionate dining in a fun and effective way.

## Tech Stack:

  <p>- ReactJs</p>
  <p>- react-router-dom</p>
  <p>- Javascript</p>
  <p>- HTML</p>
 <p>- CSS</p>

## Features:
1. Signin/Signup
2. Home Page
3. View Product Page
4. Pagination
5. search
